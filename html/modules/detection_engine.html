Manages interactions with Firestore database for vehicle detection system.

This class handles all database operations including querying vehicle information, checking for existing violations, and retrieving user data associated with license plates.

process_image(image_path, parking_data_path=None, regions=('jo',))[source]

Process an image for vehicle detection and parking analysis.

This is the main entry point for the detection pipeline, which: 1. Detects vehicles in the image 2. Recognizes license plates 3. Performs parking spot analysis if parking data is provided 4. Detects parking violations 5. Creates an annotated output image

Parameters:

  • image_path (str) – Path to the input image

  • parking_data_path (Optional[str], optional) – Path to parking spot data JSON file. Defaults to None.

  • regions (tuple, optional) – License plate region codes. Defaults to (“jo”,).

Returns:

Results dictionary containing:

  • Detected vehicles with their information

  • Parking analysis results (if applicable)

  • Path to the output image

Return type:

Dict

Raises:

FileNotFoundError – If the input image cannot be read
